# AddressManager
간단한 주소록 만들기

현재 회원들을 볼 수 있고 회원가입이 가능하며, 회원정보를 수정하거나 삭제가 가능,
회원들의 주소록을 볼 수 있으며, 주소록을 추가 할 수 있다.


구현 예정 - 주소록 수정, 삭제 기능, 로그인 기능, 회원 로그인시 회원 개인의 정보만 보이도록

#C++++ dotnet 7.0 Razor Page , Entity Framework Core, MSSQL(SQL Server) 사용
